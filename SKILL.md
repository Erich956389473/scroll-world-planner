# scroll-world-planner — 沉浸式着陆页规划器

## 它是什么

[scroll-world](https://github.com/oso95/scroll-world) 是一个 Agent Skill，它用 AI 生成那种"滚动穿越3D世界"的炫酷着陆页。但它需要品牌故事规划、场景设计和信息架构——这些正是 PM 的活。

这个 planner 帮你规划好，开发照着做就行。

## 输入

- `brand`：品牌/产品名称（必填）
- `industry`：行业（必填）
- `pitch`：核心卖点（1 句话）
- `scenes`：想要的场景数（可选，默认 4）
- `cta`：最终行动号召（可选，如"立即试用"）
- `tone`：调性（可选，默认 professional / professional / playful / luxury / minimal）

## 输出

一个完整的 scroll-world 导演脚本，开发可以直接用来生成。

## 工作流

### 第一步：品牌信息架构

分析品牌的核心信息，确定需要传达的 3-5 个关键信息点：

```
输入：某生鲜电商品牌 "鲜直达" — 30 分钟达的精品生鲜
场景数：4

信息架构：
1. 信任感——源头直采、品质保障
2. 速度感——30 分钟达的配送网络
3. 丰富度——覆盖三餐场景的品类
4. 便利性——APP 一键下单的体验
```

### 第二步：场景序列设计

为每个场景设计"飞入"路径和内容：

```
场景 1：源头（建立信任）
  视角：俯瞰产地 → 飞入田间
  画面元素：绿色田野、农户、清晨阳光
  文案叠加："从田间到餐桌，我们只选最好的"
  音乐提示：温暖、自然

场景 2：配送（速度感）
  视角：城市俯瞰 → 跟随配送员飞入街区
  画面元素：城市天际线、配送员、时间指示
  文案叠加："30 分钟，新鲜直达"
  音乐提示：节奏感、轻快

场景 3：餐桌（丰富度）
  视角：窗外 → 飞入厨房 → 停到餐桌上
  画面元素：丰富菜品、家庭用餐氛围
  文案叠加："每天不重样，三餐都精彩"
  音乐提示：温馨、生活感

场景 4：APP（便利性）
  视角：手机屏幕 → 飞入 APP 界面
  画面元素：APP 界面、下单流程、评价
  文案叠加："一键下单，坐等鲜直达"
  音乐提示：科技感、简洁
  CTA："立即体验 30 分钟送达"
```

### 第三步：品牌调性适配

**professional（专业感）：**
- 色调：蓝白灰，干净利落
- 字体：无衬线，大字号
- 节奏：稳重，每场景停留 3-4 秒

**playful（活泼感）：**
- 色调：明亮高饱和，多色彩
- 字体：圆润，大小对比强烈
- 节奏：快节奏，每场景 2-3 秒

**luxury（奢华感）：**
- 色调：金黑深色，质感光效
- 字体：衬线，优雅间距
- 节奏：缓慢，每场景 4-5 秒

**minimal（极简）：**
- 色调：黑白灰，留白多
- 字体：极简，字重轻
- 节奏：干净利落

### 第四步：输出导演脚本

最终的交付物是一个结构化的"导演脚本"：

```
## scroll-world 导演脚本

### 品牌：鲜直达
### 行业：生鲜电商
### 调性：专业感
### 场景数：4 + 1 CTA

---

### 场景 1：源头
**飞入路径：** 高空俯瞰绿色田野 → 垂直下降至田间 → 环绕农产品特写
**画面元素：**
  - 前景：新鲜蔬果（高饱和度绿色/红色）
  - 中景：农民采摘动作
  - 背景：远处的山和晨光
**文案：** "从田间到餐桌，我们只选最好的"
**配色：** #4CAF50（绿）, #8BC34A（浅绿）, #FFFDE7（米白）
**过渡：** 顺时针旋转 → 拉开至城市全景

### 场景 2：配送
**飞入路径：** 城市天际线 → 追踪配送员 → 俯冲进入社区
**画面元素：**
  - 前景：保温箱中的菜品
  - 中景：配送员骑行
  - 背景：城市建筑轮廓
**文案：** "30 分钟，新鲜直达"
**配色：** #2196F3（蓝）, #FF9800（橙）, #FFFFFF
**过渡：** 推近至门 → 开门进入室内

...

### CTA 页
**效果：** 场景 4 结束后 → 画面定格 → 品牌 logo + CTA 按钮缓入
**CTA 文案：** "立即体验 30 分钟送达"
**按钮色：** #FF5722（珊瑚橙）
**背景：** 纯色或场景 4 最后一帧渐变静帧
```
