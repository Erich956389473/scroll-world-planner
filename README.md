# scroll-world-planner

> 沉浸式着陆页的 PM 规划器 — 品牌故事 → 场景设计 → 导演脚本

[![GitHub](https://img.shields.io/badge/GitHub-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Erich956389473/scroll-world-planner)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ 功能特性

- **品牌故事线** — 构建沉浸式叙事
- **场景规划** — 自动规划滚动场景
- **调性适配** — 4 种视觉调性
- **导演脚本** — 可交付的详细脚本

## 🚀 快速开始

### 作为 Agent Skill 使用

`
scroll-world-planner
  brand: 你的品牌名
  industry: 行业
  pitch: 核心卖点
  scenes: 4
  tone: professional
`

### 调性选项

- professional — 专业感（B2B、金融）
- energetic — 活力感（运动、娱乐）
- elegant — 优雅感（奢侈品、时尚）
- 	ech — 科技感（AI、SaaS）

## 📖 输出示例

`markdown
## 导演脚本

**品牌:** 示例品牌
**调性:** 专业感
**场景数:** 4

### 场景1（开场）
**镜头:** 全景 → 特写
**文案:** 品牌核心价值
**时长:** 3s

### 场景2（产品）
**镜头:** 产品展示 → 使用场景
**文案:** 产品亮点
**时长:** 3s
`
